# Shadowing (影讀)
An app that trains your speaking via Text-To-Speech and Speech Recognition

## MVP for 1.0
### 解決的問題：
* 隨時隨地的增強口說能力 (日本語)

### Main Flow
1. 每一句 (siri 唸 -> 使用者跟著唸 -> 辨識 (辨識時播下一句話))
2. 算分
3. 統計、追蹤進度
